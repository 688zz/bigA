# 股票公司分析清单

用于在研究一只股票前，系统性了解目标公司的基本面、估值、风险和跟踪重点。

不要只写“公司有什么故事”，更要回答“哪些已经兑现、哪些还只是期权、当前估值买的是哪一段未来”。

## 使用方式

在新对话中可以这样使用：

```markdown
请读取工作区里的《股票公司分析清单.md》，并按照里面的框架分析 {公司名称}，生成一份 Markdown 股票研究报告。
```

如果目标公司属于强周期、重资产、新业务题材股，应额外要求：

```markdown
请重点区分已兑现利润、建设中项目、客户认证阶段业务和市场预期，并列出后续最适合跟踪的验证指标。
```

## 研究流程

### 1. 先定研究问题

- 这家公司当前最核心的投资问题是什么：周期修复、产能释放、技术突破、估值消化、股东回报还是困境反转
- 市场当前主要在买什么预期：主业复苏、新业务放量、行业涨价、国产替代、资产重估、并购重组等

### 2. 固定数据口径

- 报告生成时间，精确到日期；行情快照最好精确到具体时间
- 财报口径：最近年报、最近季报、是否使用 TTM 或年化数据
- 行情口径：股价、市值、PE、PB、PS、成交额、换手率的抓取来源和时间
- 财务口径：是否合并报表，是否有一次性损益、会计差错更正、资产减值、股份支付等扰动
- 来源优先级：公司公告和定期报告 > 交易所/巨潮资讯 > 官方网站 > 可靠财经数据 > 行业公开报道 > 市场传闻

### 3. 做交叉检查

- 关键财务指标至少和年报、季报或可靠数据源交叉验证一次
- 动态 PE、TTM PE、静态 PE 要分清，不用单一 PE 下结论
- 对强周期公司，不直接把单季利润年化当作稳定盈利能力
- 对新业务题材，不把“项目建成”“取得认证”“试生产”直接等同于“稳定盈利”
- 对热门题材，检查收入占比、毛利率、子公司利润和订单规模是否支撑市场叙事

## 分析框架

### 1. 公司基本信息

- 公司全称、股票简称、股票代码
- 所属市场：A股 / 港股 / 美股 / 其他
- 所属板块：例如白酒、半导体、新能源、银行、医药、互联网等
- 投资周期：短线 / 中线 / 长期
- 风险偏好：低 / 中 / 中高 / 高
- 核心业务：当前主要收入和利润来源
- 潜在业务：尚未大规模贡献收入或利润的新业务、储备业务、第二曲线
- 报告生成时间、关键数据口径和来源日期

### 2. 核心结论先行

- 一句话说明公司当前是什么类型：稳定现金流、周期弹性、成长兑现、题材期权、困境反转等
- 一句话说明当前最大矛盾：好公司但贵、故事大但兑现少、估值低但周期弱、资产好但治理复杂等
- 初步结论：偏乐观 / 中性偏积极 / 中性 / 中性偏谨慎 / 偏谨慎

### 3. 公司做什么

- 主营业务是什么
- 产品或服务卖给谁
- 收入主要来自哪些业务线，各业务收入占比和毛利率是多少
- 利润主要来自哪些业务线，是否和收入结构一致
- 公司处在产业链的哪个位置：上游资源 / 中游制造 / 下游品牌 / 平台服务 / 配套材料
- 业务是否依赖单一客户、单一供应商、单一项目或单一价格周期

### 4. 行业前景与周期位置

- 行业是在增长、成熟、衰退还是周期修复
- 市场空间有多大，增量来自哪里
- 行业竞争是否激烈，价格战是否明显
- 是否受政策、技术周期、经济周期、商品价格、利率或汇率影响很大
- 当前处在行业周期的什么位置：底部修复、景气上行、高位回落、库存去化、产能出清等
- 对周期行业，要同时分析需求、供给、库存、价格和企业成本位置

### 5. 商业模式

- 公司怎么赚钱：卖产品、收服务费、资源开采、项目制、设备销售、加工费、投资收益等
- 收入是否稳定、可重复，客户是否有复购
- 毛利率高不高，毛利率变化来自价格、成本、产品结构还是产能利用率
- 是否有规模效应、网络效应、品牌效应、渠道优势、资源禀赋、客户认证或牌照壁垒
- 是否重资产，资本开支、折旧、在建工程转固会不会压制利润
- 现金流是否天然好，还是容易被应收账款、存货和项目回款占用

### 6. 竞争力

- 相比同行有什么优势
- 市占率如何，是否有全球或国内排名
- 客户粘性强不强，客户认证周期有多长
- 是否有技术、专利、品牌、成本、渠道、牌照、资源储量或产能规模等护城河
- 竞争力是否能转化为毛利率、净利率、ROE 和现金流，而不只是规模领先
- 与主要同行相比，公司强在哪里，弱在哪里

### 7. 财务状况

- 营收和利润是否持续增长，增长来自销量、价格、产品结构还是并表
- 毛利率、净利率、ROE 是否健康，和同行相比处在什么水平
- 经营现金流是否充足，是否持续高于或低于净利润
- 资产负债率、短债、货币资金是否匹配资本开支需求
- 应收账款、存货、合同资产、在建工程、商誉是否异常增长
- 是否有资产减值、存货跌价、信用减值、会计差错更正、非经常性损益或股份支付费用
- 对重资产公司，单独看资本开支、折旧摊销、产能利用率和项目转固节奏

### 8. 估值水平

- 当前股价、市值、流通市值是多少，行情快照时间是什么
- PE、PB、PS、PEG、股息率等指标分别适不适合这家公司
- 静态 PE、TTM PE、动态 PE、单季年化 PE 是否差异很大，差异原因是什么
- 相对历史估值如何，相对同行估值如何
- 市值是否匹配未来成长空间和已兑现利润
- 市场预期是否已经过高：当前估值是在买主业、周期修复，还是买尚未兑现的新业务
- 若估值偏高，需要说明后续需要什么业绩才能消化估值

### 9. 增长来源

- 未来增长靠什么：销量、价格、产能、客户、产品升级、成本下降、并购、海外市场、政策红利等
- 增长是内生经营，还是靠并购、涨价、补贴、周期红利或会计因素
- 增长的确定性排序：已投产放量 > 建设后期 > 客户认证 > 试验线/研发 > 概念储备
- 增长是否能转化为利润和现金流，还是只带来收入和资本开支
- 新业务和原主业是否协同，是否存在跨界扩张风险

### 10. 潜在业务与项目管线

- 公司有哪些尚未大规模贡献收入或利润的新业务、储备业务、第二曲线
- 潜在业务处于什么阶段：概念 / 研发 / 试验线 / 试生产 / 小批量 / 客户认证 / 量产爬坡 / 稳定盈利
- 潜在业务收入占比、毛利率、子公司净利润、订单和客户情况是否披露
- 潜在业务与公司原有能力是否协同，还是跨界扩张
- 市场空间、竞争格局、技术门槛和客户验证难度如何
- 资本开支、研发投入、折旧、亏损或减值风险有多大
- 市场是否已经把潜在业务预期计入估值

### 11. 管理层与公司治理

- 管理层是否稳定，核心高管履历是否匹配主业
- 实控人、控股股东、董事会结构是否清晰
- 是否有长期战略能力，过去承诺的项目有没有兑现
- 是否重视股东回报，分红、回购、注销、股权激励安排如何
- 大股东和管理层是否有频繁减持、质押、资金占用、关联交易、违规担保等风险
- 是否存在会计差错更正、监管函、处罚、诉讼、审计保留意见等披露质量问题

### 12. 股东回报与融资稀释

- 是否分红，分红是否稳定，分红率和股息率是否有吸引力
- 是否回购，回购是注销还是用于股权激励
- 是否频繁融资、增发、可转债或配股，是否稀释股东权益
- 分红、回购、资本开支、债务压力之间是否平衡
- 对成长股和项目建设期公司，要特别看未来是否还需要继续融资

### 13. 风险因素

- 政策监管风险
- 行业周期风险
- 技术替代风险
- 客户集中和供应商集中风险
- 原材料、能源、商品价格和库存跌价风险
- 汇率、利率、地缘政治和贸易限制风险
- 安全生产、环保、危险化学品、矿山开采等合规风险
- 财务造假、治理、质押、减持、关联交易和信息披露风险
- 估值过高和题材退潮风险

### 14. 市场情绪与资金面

- 当前是否处在热门题材中，题材和主业的关系有多强
- 成交额、换手率、流动性是否明显放大
- 机构持仓、北向资金、公募、险资或游资关注度如何
- 短期股价是否已经透支利好
- 是否存在事件催化：业绩预告、项目投产、中报、产品认证、涨价、政策、行业会议等

### 15. 信息披露与历史记录

- 年报、季报是否清晰透明，分部披露是否足够
- 公司是否经常变更口径、讲故事、频繁跨界
- 过去承诺的项目、产能、客户、订单有没有兑现
- 是否有处罚、诉讼、审计保留意见、监管问询、会计差错更正等问题
- 信息披露是否能支持投资逻辑，还是只能支持题材想象

### 16. 跟踪指标与验证清单

- 列出 5-10 个最适合继续跟踪的关键指标
- 每个指标都要说明“为什么重要”
- 区分经营指标、财务指标、行业指标、项目指标和资金面指标
- 把需要进一步验证的信息单独列出，避免把不确定信息写成结论
- 对下一期财报或下一次公告，提前写清楚应该重点看什么

## 行业适配检查项

### 强周期与资源股

- 核心商品价格：现货、期货、长协价、加工费
- 产量、销量、单位售价、单位成本
- 资源储量、品位、采矿权、采选能力、扩产项目进度
- 库存、存货跌价准备、应收账款回款
- 行业供需、库存周期、产能出清和成本曲线
- 不把高景气单季利润直接年化为长期利润

### 重资产制造

- 产能、产能利用率、良率、稼动率、折旧
- 在建工程转固节奏和新增折旧压力
- 经营现金流与资本开支的差额
- 客户导入、订单持续性、产品价格和毛利率
- 新产线从建成到达产再到盈利的时间差

### 半导体材料、电子特气和高端制造

- 产品纯度、良率、稳定性、客户认证和复购情况
- 收入占比、毛利率、子公司利润和订单规模
- 国产替代空间是否真实，还是只停留在概念
- 外资巨头和国内同行竞争格局
- 认证通过不等于批量销售，批量销售不等于稳定盈利

### 高股息与稳定现金流公司

- 分红连续性、分红率、股息率和自由现金流覆盖
- 负债率、资本开支、再融资需求
- 盈利是否稳定，是否靠一次性收益支撑分红
- 回购是否注销，是否真正提升每股价值

## 报告输出建议

- 报告标题：`# {股票代码}{公司名称}_{生成日期}`
- 文件命名：`{股票代码}{公司名称}_{生成日期}.md`
- 报告声明：公开资料整理，不构成买卖建议
- 公司基本信息表
- 核心结论与最大矛盾
- 公司做什么
- 行业前景与周期位置
- 商业模式
- 竞争力
- 关键财务指标表
- 估值快照与同业估值对比表
- 增长来源
- 潜在业务/项目管线分析表
- 管理层与公司治理
- 股东回报与融资稀释
- 主要风险
- 市场情绪与资金面
- 信息披露与历史记录
- 适合继续跟踪的关键指标表
- 需要进一步验证的信息
- 初步结论：偏乐观 / 中性偏积极 / 中性 / 中性偏谨慎 / 偏谨慎
- 数据来源

## 质量标准

- 明确区分事实、判断、推测和需要进一步验证的信息
- 优先使用最新公开资料、财报、公告和可靠财经数据
- 标注关键数据来源、日期和行情快照时间
- 不直接给出买入或卖出建议
- 信息不足时，列出需要补充查询的问题
- 对热门新业务保持克制：先看收入、利润、现金流和订单，再谈估值重构
- 对周期股保持克制：先看价格周期、成本位置和现金流，再谈利润弹性
- 对高估值公司说明“需要怎样的兑现才能支撑当前市值”

## 常见误区

- 把“公司有某项业务”当作“该业务已经贡献利润”
- 把“项目达到可使用状态”当作“项目已经达产盈利”
- 把“客户认证通过”当作“客户已经批量采购”
- 把“单季高利润”当作“全年和长期稳定利润”
- 只看 PE，不看 PB、PS、现金流、周期位置和利润质量
- 只看题材热度，不看收入占比、毛利率和子公司净利润
- 只写投资亮点，不写估值已经反映了多少预期

## 一句话总结

先看公司是不是好生意，再看是不是好公司，最后看现在是不是好价格；对题材股和周期股，还要多问一句：现在的价格买的是现实，还是未来。
