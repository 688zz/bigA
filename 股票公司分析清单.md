# 股票公司分析清单

用于在研究一只股票前，系统性了解目标公司的基本面、估值、风险和跟踪重点。

## 使用方式

在新对话中可以这样使用：

```markdown
请读取工作区里的《股票公司分析清单.md》，并按照里面的框架分析 {公司名称}，生成一份 Markdown 股票研究报告。
```

## 分析框架

### 1. 公司基本信息

- 公司全称、股票简称、股票代码
- 所属市场：A股 / 港股 / 美股 / 其他
- 所属板块：例如白酒、半导体、新能源、银行、医药、互联网等
- 投资周期：短线 / 中线 / 长期
- 风险偏好：低 / 中 / 高
- 核心业务：当前主要收入和利润来源
- 潜在业务：例如 AI 应用、机器人、玻璃基板、固态电池、海外业务、第二增长曲线等
- 报告生成时间、关键数据口径和来源日期

### 2. 公司做什么

- 主营业务是什么
- 产品或服务卖给谁
- 收入主要来自哪些业务线
- 公司处在产业链的哪个位置

### 3. 行业前景

- 行业是在增长、成熟还是衰退
- 市场空间有多大
- 行业竞争是否激烈
- 是否受政策、技术周期、经济周期影响很大

### 4. 商业模式

- 公司怎么赚钱
- 收入是否稳定、可重复
- 毛利率高不高
- 有没有规模效应、网络效应、品牌效应或渠道优势

### 5. 竞争力

- 相比同行有什么优势
- 市占率如何
- 客户粘性强不强
- 是否有技术、专利、品牌、成本、渠道、牌照等护城河

### 6. 财务状况

- 营收和利润是否持续增长
- 毛利率、净利率、ROE 是否健康
- 现金流是否充足
- 负债率是否过高
- 应收账款、存货是否异常增长

### 7. 管理层与公司治理

- 管理层是否靠谱、稳定
- 是否有长期战略能力
- 是否重视股东回报
- 大股东和管理层是否有频繁减持、质押等风险

### 8. 估值水平

- 当前股价贵不贵
- PE、PB、PS、PEG 等指标相对历史和同行如何
- 市值是否匹配未来成长空间
- 市场预期是否已经过高

### 9. 增长来源

- 未来增长靠什么
- 新产品、新市场、新客户是否真实可持续
- 增长是靠内生经营，还是靠并购、涨价、补贴、周期红利

### 10. 潜在业务

- 公司有哪些尚未大规模贡献收入或利润的新业务、储备业务、第二曲线
- 潜在业务处于概念、研发、试验线、小批量、客户验证、量产爬坡还是规模化盈利阶段
- 潜在业务与公司原有能力是否协同，还是跨界扩张
- 市场空间、竞争格局、技术门槛和客户验证难度如何
- 资本开支、研发投入、折旧、亏损或减值风险有多大
- 市场是否已经把潜在业务预期计入估值

### 11. 风险因素

- 政策监管风险
- 行业周期风险
- 技术替代风险
- 客户集中风险
- 原材料价格风险
- 汇率、利率、地缘政治风险
- 财务造假或治理风险

### 12. 股东回报

- 是否分红
- 分红是否稳定
- 是否回购
- 融资、增发是否频繁稀释股东权益

### 13. 市场情绪与资金面

- 当前是否处在热门题材中
- 机构持仓变化如何
- 成交量和流动性是否足够
- 短期股价是否已经透支利好

### 14. 信息披露与历史记录

- 年报、季报是否清晰透明
- 公司是否经常变更口径、讲故事
- 过去承诺的目标有没有兑现
- 是否有处罚、诉讼、审计保留意见等问题

## 报告输出建议

生成 Markdown 股票研究报告时，建议包含以下内容：

- 报告标题：`# {股票代码}{公司名称}_{生成日期}`
- 文件命名：`{股票代码}{公司名称}_{生成日期}.md`
- 公司基本信息
  - 股票代码
  - 所属市场
  - 所属板块
  - 投资周期
  - 风险偏好
  - 核心业务
  - 潜在业务
  - 报告生成时间
- 关键财务指标表
- 估值对比表
- 投资亮点
- 主要风险
- 潜在业务分析
- 适合继续跟踪的关键指标
- 需要进一步验证的信息
- 初步结论：偏乐观 / 中性 / 偏谨慎

## 分析原则

- 明确区分事实、推测和需要进一步验证的信息
- 优先使用最新公开资料、财报、公告和可靠财经数据
- 标注关键数据来源和日期
- 不直接给出买入或卖出建议
- 信息不足时，列出需要补充查询的问题

## 一句话总结

先看公司是不是好生意，再看是不是好公司，最后看现在是不是好价格。
